# CSS Grid (school challenge)


### Task list:

```
• Create an HTML page that has one <input> type number, which receives numbers from 2-50, 
  one <select> with 3 colors options- red, blue and green, and one <button> - run.

• Write a code that by clicking the "run" button for the entered integer x draw x*x number of squares
  in the selected color, across square shape div. 

• The diagonal from the upper left, to the lower right, should always be painted in yellow.
```
